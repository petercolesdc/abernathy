Abernathy: Pattern library & flat-site building blocks

Abernathy is a place to start product and web builds. Built with atomic component based systems in mind, she renders to standard html templates, so you can build in a modular fashion – then render the old fashioned way.

Abernathy uses the Nunjucks templating language to harness the power of includes, macros and loops to aid in building dynamic design systems.

Checkout the README for help on how to get started.
